# Research Complete Checklist

## Market Research
- [ ] Market size estimated
- [ ] Growth trend identified
- [ ] Key demand signals documented
- [ ] Market opportunity scored

## Competitor Research
- [ ] At least 3 competitors identified
- [ ] Competitive gaps documented
- [ ] Positioning defined

## User Research
- [ ] Target user profiled
- [ ] Pain points validated
- [ ] User needs summarized

## Problem Research
- [ ] Core problem articulated
- [ ] Root cause identified
- [ ] Why existing solutions fail — documented

## Monetization Research
- [ ] Revenue models evaluated
- [ ] Best fit model selected
- [ ] First revenue experiment defined

## Technical Research
- [ ] Required features listed
- [ ] Integrations identified
- [ ] Build complexity assessed
- [ ] Technical unknowns documented

## Opportunity Summary
- [ ] `opportunity-summary.md` complete
- [ ] Recommendation is Proceed / Refine / Pause
- [ ] User reviewed recommendation

## Gate Result

- [ ] Research approved — proceed to Planning
